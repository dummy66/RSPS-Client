Client-side

This is going to be an attempt to dissect each class and identify its function in the client.
First and foremost, the most essential classes are Gui.class and client.class. These are the
"control centers" of the game engine. These two files contain game information with all other
classes intertwined.


Applet_Sub1.class
-----------------
This is the class the "client.class" extends from. It is the base.
This class ultimately extends from the Applet library of Java. It was customized and changed
for RSPS.
All base GUI information is here (prior to GUI initialization).
